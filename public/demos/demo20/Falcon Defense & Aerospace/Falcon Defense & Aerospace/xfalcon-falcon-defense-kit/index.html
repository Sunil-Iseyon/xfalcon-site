Contract portfolio health uses PERIOD_OF_PERF_END ≥ CURRENT_DATE as the "active contract" definition (DIM_CONTRACT has no IS_ACTIVE flag). All dollar values are nominal USD, displayed in millions ($M) unless noted. This is a demo environment — all classification, agency, and program data is synthetic.
