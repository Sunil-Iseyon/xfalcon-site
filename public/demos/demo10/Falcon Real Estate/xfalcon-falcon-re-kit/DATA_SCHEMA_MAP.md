# Falcon Real Estate — Data Schema Map

**Database:** Apex REIT Portfolio
**Schema Type:** Star schema with shared dimension hub (DIM_PROPERTY)
**Date Range:** 2020-01-01 to 2025-12-31 (72 months)

## Key Join Paths

```
DIM_PROPERTY (hub)
├── → DIM_ASSET_CLASS    via ASSET_CLASS_ID
├── → DIM_MARKET         via MARKET_ID
├── → DIM_PROPERTY_MANAGER via MANAGER_ID
├── → FACT_NOI           via PROPERTY_ID (+ MONTH_ID → DIM_TIME)
├── → FACT_OCCUPANCY     via PROPERTY_ID (+ MONTH_ID → DIM_TIME)
├── → FACT_RENT_COLLECTION via PROPERTY_ID (+ MONTH_ID → DIM_TIME)
├── → FACT_LEASE         via PROPERTY_ID
├── → FACT_MAINTENANCE_WO via PROPERTY_ID (+ REPORTED_DATE_ID → DIM_TIME)
├── → FACT_CAPEX_PROJECT via PROPERTY_ID
└── → FACT_TENANT_RENEWAL via PROPERTY_ID

FACT_RENT_COLLECTION
├── → FACT_LEASE via LEASE_ID
└── → DIM_TENANT via TENANT_ID

FACT_LEASE
├── → DIM_TENANT via TENANT_ID
└── → DIM_LEASE_TYPE via LEASE_TYPE_ID

FACT_MARKET_BENCHMARK
├── → DIM_MARKET via MARKET_ID
├── → DIM_ASSET_CLASS via ASSET_CLASS_ID
└── → DIM_TIME via QUARTER_ID
```
