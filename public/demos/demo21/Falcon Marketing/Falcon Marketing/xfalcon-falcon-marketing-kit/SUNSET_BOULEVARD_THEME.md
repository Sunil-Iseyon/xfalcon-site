# Sunset Boulevard Theme — Falcon Marketing (alternate)

Secondary theme for the runtime switcher. Warm-creative mood — burnt orange, coral, sand on cream.

## Palette

| Role | Token | Hex |
|------|-------|-----|
| Topbar | `--color-topbar` | `#264653` |
| Body bg | `--color-bg` | `#FEF7F0` |
| Card bg | `--color-card` | `#FFFFFF` |
| Card border | `--color-border` | `#F0E4D7` |
| Primary | `--color-primary` | `#E76F51` |
| Secondary | `--color-secondary` | `#F4A261` |
| Tertiary | `--color-tertiary` | `#E9C46A` |
| Text primary | `--color-text` | `#1C1917` |
| Text muted | `--color-text-muted` | `#6B5D4F` |
| Text on dark | `--color-text-on-dark` | `#FEF7F0` |
| Positive | `--color-positive` | `#059669` |
| Negative | `--color-negative` | `#D32F2F` |
| Neutral | `--color-neutral` | `#94A3B8` |

Status colors and year-palette indices (CY/PY/PPY) stay unchanged from Midnight Galaxy — theme affects chrome only, not chart data semantics.

## Logo

`xF` — `x` in `#E76F51`, `F` in `#F4A261`.

## Notes

Activated via `?theme=sunset` URL parameter. The runtime switcher is wired up in `theme.css` and `theme.js`; no separate dashboard files needed.
